Visão Geral do Sistema

Arquitetura em 7 camadas do pipeline completo de detecção de fraudes

Como funciona — em linguagem simples Negócio

📥
1. Coletar
Dados de pedidos, motoristas, clientes e produtos são importados automaticamente
🧹
2. Limpar
Os dados são validados, padronizados e armazenados de forma segura
🔍
3. Analisar
Algoritmos identificam padrões anômalos por motorista, cliente e região
🚨
4. Pontuar & Alertar
Cada entidade recebe uma pontuação de risco (0–100) e é categorizada
📊
5. Visualizar
Dashboard interativo com 9 painéis para monitoramento contínuo
Resultado: O Walmart consegue identificar motoristas e clientes suspeitos antes que as perdas escalem, priorizando investigações onde o risco é maior.

Arquitetura de Dados

5 arquivos CSV formam o modelo relacional completo do sistema

Nossas fontes de dados Negócio

📦
orders.csv
~8.500
Tabela principal: cada linha é um pedido entregue. Contém valor, região, itens entregues e itens faltantes.
🚗
drivers_data.csv
~1.200
Informações dos motoristas: nome, idade e total de viagens em 2023.
👤
customers_data.csv
~1.000
Cadastro de clientes: nome e idade.
❌
missing_items_data.csv
~3.000
Produtos reportados como não recebidos — até 3 produtos por pedido.
🏷️
products_data.csv
~500
Catálogo de produtos com nome, categoria e preço.
Todos os arquivos convergem para o banco de dados central

04

Modelos de Machine Learning

4 algoritmos + engine de pontuação composta com pesos calibrados

Como detectamos fraudes Negócio

🔎
Detecção de Anomalias
Dois algoritmos identificam pedidos, motoristas e clientes que fogem do comportamento normal — como um segurança que percebe alguém agindo de forma suspeita na multidão.
🗂️
Agrupamento (Clustering)
Os dados são organizados em grupos similares. Um grupo com alta taxa de itens faltantes é sinalizado como de alto risco para investigação prioritária.
⚖️
Pontuação de Risco
Todos os sinais são combinados numa nota de 0 a 100. Quem está acima de 75 é marcado como Crítico e entra na fila de investigação imediata.
0–24
Baixo Risco
25–49

50–74

Crítico
