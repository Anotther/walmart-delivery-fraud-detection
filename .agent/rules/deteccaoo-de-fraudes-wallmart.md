Detecção de Fraudes em Entregas Walmart E-commerce Projeto de ciência de dados para identificar padrões de fraude em entregas online do Walmart na região central da Flórida. O foco é analisar dados de 2023 para detectar se itens reportados como "não recebidos" pelos clientes são resultado de fraude de entregadores, consumidores ou falhas sistêmicas. Contexto: Walmart perdeu $6,5 bilhões em 2023 com roubos, sendo 53% do crescimento atribuído ao e-commerce. Este projeto piloto pode ser expandido para todo EUA se bem-sucedido. Entregáveis: Relatório completo, dashboard de monitoramento, recomendações preventivas e propostas de melhorias futuras.
