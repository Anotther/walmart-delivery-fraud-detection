## Repository Snapshot
- `src/`: ETL, features, models, analysis, database, dashboard utilities.
- `dashboard/`: Streamlit app entrypoint, pages, and UI components.
- `scripts/`: setup, ETL, model training, and reporting commands.
- `data/`: source CSV datasets for 2023 operations.
- `tests/`: pytest suites and dashboard audits.
- `docs/`: business and product documentation.
